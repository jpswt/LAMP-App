API Server

GET /chemicals

This route will return an array for each chemical in our app. All details are included for the chemical: id, chemical name, CAS number and manufacturer

GET /chemicals/:id

This route will return an array for a single chemical given its id in our app. Only the required details for for the chemical: id, chemical name, CAS number are included.

POST /chemicals

This route is used to create a new chemical item. In order to create a new chemical, both a chemical name and CAS number must be provided in the body of the request. The manufacturer information is optional(can be NULL).

{"chemicalName":"Test Chemical", "casNumber":"123-456-78"}

PUT /chemicals/:id

This route is used update a chemical item given an id. In order to create a new chemical, both a chemical name and CAS number must be provided in the body of the request. The manufacturer information is optional(can be NULL).

{"chemicalName":"Test Chemical", "casNumber":"876-53-09"}

DELETE /chemicals/:id

This route is to used delete a chemical item given an id. It will give a 204 response to let the client know that it was chemical item was deleted.
